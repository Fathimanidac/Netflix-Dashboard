
"Netflix Movies and TV shows Dashboard"

I created an interactive Netflix I created an interactive Netflix Dashboard using Power BI to analyze and visualize Netflix content data in a clear and user-friendly format. The dashboard provides a complete overview of Netflix’s catalog, showing key insights such as total titles, ratings count, genres, release years, countries, and content types (Movies vs TV Shows).

The report includes KPI cards displaying important metrics like Total Titles (5332), Total Ratings, Total Genres, Start Year (1942), End Year (2021), and Total Locations (604). It also contains visualizations such as genre distribution bar charts, rating-wise title comparison, Movies vs TV Shows donut chart, Top 10 countries treemap, and release-year trend line chart to identify growth patterns over time.

This dashboard helps users quickly understand Netflix content trends, popular genres, rating distribution, and country-wise production contribution. It is designed with a clean theme and interactive visuals for better decision-making and storytelling.
